一、项目1 是什么？

Mini-Drop = 在两周内**独立复刻**腾讯内部 Drop 性能分析平台的精简版。

一句话：Linux 上部署 Agent，Web 上点一下采集某进程 CPU/内存/IO，数据上传后生成火焰图、热点分析；可选 LLM 智能归因。

四组件架构（官方复刻指南）：
  drop/           C++ — Agent + Server，gRPC 心跳与任务下发
  apiserver/      Go  — REST API、PostgreSQL、对象存储（MinIO 替代 COS）
  analysis/       Py  — perf.data → 火焰图 SVG / TopN / 规则建议
  web_frontend/   React — 建任务、轮询状态、看火焰图

验收标准：`docker compose up` 一键启动 → Web 建任务 → 看到火焰图。
