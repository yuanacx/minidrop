二、能否在 Linux 运行

| 平台 | 结论 |
|------|------|
| Windows `.exe` | 仅 Windows；**不能**在 ECS Linux 运行 |
| Linux ELF | ✅ ECS 上可执行、可心跳注册 |

Windows 二进制不能替代 Linux 部署；生产必须在 Linux 上 `bash drop/build_agent.sh` 构建。

────────────────────────────────────────

三、ECS 心跳自测

```bash
docker compose stop drop_agent
DROP_SERVER=http://127.0.0.1:50051 APISERVER=http://127.0.0.1:8191 AGENT_ID=agent-bin \
  timeout 12 /opt/minidrop/bin/drop_agent
curl http://127.0.0.1:50051/control/list_agents
# → online:true, last_seen 更新
/opt/minidrop/bin/drop_agent --version  # drop_agent 0.1.0
docker compose start drop_agent
```

**结果：心跳注册成功。**

说明：裸机运行时 CP 线程可能报 `perf: No such file or directory`（宿主机未装 linux-tools），与 Docker 容器内不同；心跳与任务调度不受影响。perf/py-spy/bpftrace 仍需宿主机 PATH 安装，PyInstaller 不打包这些外部工具。

五、py-spy 打包说明

py-spy 是**独立 Rust 二进制**，非 Python 模块，无法用 PyInstaller 打入 drop_agent。替代：ECS 上 `pip install py-spy` 或 apt 安装，保证 PATH 有 `py-spy` 即可。
